# SIGSM — Sistema de Información y Gestión para Salud y Medicina

> Proyecto integrador UTU 2026 — Hospital de Clínicas
> Equipo **QuadNet** · ISBO

---

## Descripción

SIGSM es un sistema de gestión hospitalaria desarrollado para el Hospital de Clínicas como proyecto integrador del curso de Informática de UTU (2026). Permite administrar pacientes, consultas médicas, personal y recursos del hospital a través de una interfaz web.

**Primera entrega:** junio 2026
